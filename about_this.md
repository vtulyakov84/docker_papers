В связи с тем, что автором репозитория используется ОС Alt Linux, вся представленна информация апробирована именно на этой ОС. При использовании других ОС изменения будут не значительны, например, `$ docker-compose ..` или `$ docker compose ...`.

Создание репозитория обусловлено следующими обстоятельствами:
1. Повышения квалификации по работе с микросервисными архитектурами. 
2. Разработка программного обеспечения для участия в конкурсе лучший по профессии, проводимого в ПАО  "Россети-Юг"
3. Обобщение разнообразия результатов поиска информации в сети Интернет.

В связи с тем, что отрасль энергетики имеет отношение к работе с объектами критической информационной инфраструктуры (КИИ) родительским образом, также рассматривается работа с родительским образом `alt:p10` при оформлении Dockerfile.
